Projeto de extensão: Maratona de Programação


Funcionamento

Objetivo:
Treinar alunos para a participação em maratonas de programação e promover uma maratona envolvendo alunos do grupo Anima.

Organização:
- A primeira fase deste projeto consiste em um treinamento para maratonas de programação.

- A linguagem de programação utilizada no treinamento é Python 3.

- O ambiente utilizado é o BOCA, o mesmo utilizado pela maratona brasileira. Ele está executando na Oracle Cloud e pode ser encontrado aqui: http://maratona.rodrigobossini.com.br/boca/

- O ambiente BOCA lida com as linguagens C, C++, Java e Python.

- Durante o treinamento, cada aluno participa de maneira individual. No dia da maratona, os alunos participarão em grupos de dois a três membros.

- Para que os alunos possam ser cadastrados no BOCA, eles devem preencher o seguinte Google Form. https://bit.ly/maratona_cadastro_individual_participantes

- O prazo para preenchimento é dia 27/03/2022.

- O primeiro encontro para treinamento ocorrerá no dia 06/04/2022.

- Todos os encontros para treinamento ocorrerão das 20h às 21h.

- As datas e conteúdos previstos são

06/04 (quarta)
13/04 (quarta)
20/04 (quarta)
27/04 (quarta)
04/05 (quarta)
11/05 (quarta)
18/05 (quarta)
29/05 (domingo) (pode sofrer alterações, fique atento)

As condições para que o aluno seja considerado aprovado no projeto e, portanto, faça jus às horas de extensão são:

- Durante o treinamento, para cada problema proposto, fazer pelo menos uma submissão, ainda que não aceita.

- Durante o treinamento, ter, pelo menos, 70% dos problemas resolvidos e aceitos pelo BOCA.

- Participar da Maratona no dia especificado.

- Ter, pelo menos, 70% de presença nos dias de treinamento com aulas ao vivo.

- O controle de presença será realizado por meio do Google Form a seguir. O preenchimento deve ser feito no dia e horário da respectiva aula.

- O professor disponibiliza um grupo no Discord por meio do qual os alunos podem esclarecer dúvidas.
